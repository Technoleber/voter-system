# Vote Eligibility System

A simple Python-based project that checks whether a person is eligible to vote based on their age.

## Features

- Takes user age as input
- Checks voting eligibility
- Displays appropriate messages
- Simple conditional logic implementation
- Beginner-friendly Python project

## Technologies Used

- Python

## How It Works

The system:
1. Accepts age input from the user
2. Verifies whether the age is 18 or above
3. Displays eligibility status

## Example

```python
Enter your age: 20
Yes, you can vote
```

```python
Enter your age: 15
You are not eligible
```

## Concepts Used

- Variables
- User Input
- Conditional Statements (`if`, `elif`, `else`)
- Comparison Operators
